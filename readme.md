## 🇵🇱 Polski

Minimalistyczna webapp do nauki z fiszek. Wgrywasz `.csv` / `.txt` / `.tsv` (dwie kolumny), oceniasz *Umiem ✓* / *Nie umiem ✗*, na końcu powtarzasz nieumiane.

- Automatyczne renderowanie wzorów (KaTeX, też eksporty z NotebookLM)
- Tryb losowy, pełny ekran, motyw jasny/ciemny/auto
- Swipe, klawiatura (`◄ ►`, `1`/`2`, spacja)
- Talia zapisywana lokalnie, plik nigdzie nie jest wysyłany

---

## 🇬🇧 English

Minimalist flashcard webapp. Upload `.csv` / `.txt` / `.tsv` (two columns), rate *Got it ✓* / *Don't know ✗*, then retry the missed ones.

- Auto math rendering (KaTeX, NotebookLM exports supported)
- Shuffle, fullscreen, light/dark/auto theme
- Swipe, keyboard (`◄ ►`, `1`/`2`, space)
- Deck saved locally, files never uploaded anywhere
